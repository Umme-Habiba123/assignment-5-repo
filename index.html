Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an element. For example, if you want to select an element by ID, you should use "getElementById", and if you want to select an element by class, you can use "getElementsByClassName".
Question-2: What is the difference between innerHTML, innerText, and textContent ?

What I understand is that "innerHTML" can select the entire HTML inside an element and can change the full HTML content. On the other hand, "innerText" only selects and retrieves the text or value inside the element.
Question-3: What is event delegation in the DOM?

In the DOM, event delegation is a technique where you use a single parent element to handle events for all of its child elements, and you only need to use one event listener instead of adding separate event listeners for each child element.
Question-4: What is event bubbling in the DOM?

In the DOM, event bubbling is when we add an event, it first triggers on the target element, and then slowly moves up towards the parent elements, eventually reaching the document.
